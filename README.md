# 專業 BMI 智能計算機（Windows Forms 版）

這是一個基於 **C# Windows Forms** 開發的 BMI（身體質量指數）計算應用程式。  
本程式提供直覺化的圖形介面，並依據衛福部國民健康署的標準進行六段式體位判定，協助使用者快速掌握自身健康狀況。

---

## 一、專案特色

- 使用 **C# Windows Forms** 開發
- 提供簡潔直觀的操作介面
- 支援滑鼠操作與鍵盤快捷輸入
- 可自動計算 BMI 指數
- 根據 BMI 結果進行六段式體位分類
- 自動切換結果面板顏色與對應示意圖片

## 三、視窗操作說明

本程式提供滑鼠與鍵盤兩種操作方式：

### 1. 使用滑鼠操作

- 在左側「身高 (cm)」欄位點擊並輸入數值
- 在左側「體重 (kg)」欄位點擊並輸入數值
- 點擊下方「計算 BMI」按鈕即可產出結果

### 2. 使用鍵盤快捷操作

#### Enter 鍵跳轉

- 在「身高」欄位輸入完畢後，按下 `Enter`
- 游標會自動跳轉至「體重」欄位

- 在「體重」欄位輸入完畢後，按下 `Enter`
- 程式將直接執行計算並顯示結果

#### Tab 鍵切換

- 使用鍵盤上的 `Tab` 鍵，可以依序在以下元件之間切換焦點：
  - 身高輸入框
  - 體重輸入框
  - 計算按鈕
